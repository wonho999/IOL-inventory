# IOL 재고관리 시스템

안과 클리닉의 백내장 수술용 IOL(인공수정체) 렌즈를 도수별로 관리하는 웹 기반 재고관리 시스템.

## 빠른 실행

이 저장소의 `index.html` 파일을 브라우저에서 열기만 하면 됩니다. 설치·서버 불필요.

GitHub Pages가 활성화되어 있다면 다음 URL로 어디서든 접근 가능:
```
https://<GitHub사용자명>.github.io/iol-inventory/
```

## 주요 기능

- **도수별 재고 그리드**: 단초점/다초점은 5D 단위 그룹, 토릭은 CYL별 그룹
- **3색 상태 표시**: 🟢 충분 / 🟡 주문필요 / 🔴 재고없음
- **수술 일정 연동**: 오늘 수술 예정 도수에 파란 점, 수술 후 재고 부족 시 빨간 경고
- **빠른 입력**: `20.0D 3` / `18.5 +1.50 2` / `-1` 형식 텍스트 입력
- **스캔 입력**: 여러 줄 한번에 처리 (바코드 리더 호환)
- **그룹 접기/펼치기**: 모델별 상태 기억 (localStorage)
- **자동 셀 크기**: 화면 크기에 따라 auto-fill, +/- 줌 가능
- **화면 텍스트 편집**: 모든 라벨·버튼·문구 자유 수정 (병원명 변경 등)
- **주문서 출력 / Excel 내보내기**

## 데이터 저장

브라우저 `localStorage`에 저장됩니다. 키 목록:
- `iol3_state` — 모델·재고·일정
- `iol3_labels` — 화면 텍스트 커스터마이징
- `iol3_gopts` — 줌·기준값 등 옵션
- `iol3_collapse` — 그룹 접기/펼치기 상태

데이터 백업/복원: ⚙ 설정 → 💾 데이터 탭에서 JSON 내보내기/불러오기 가능.
